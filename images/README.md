# images/

Empty as of this writing. Reserved for hardware photographs (PCB, LED
package, board revision markings) — none were taken during this project,
which relied entirely on software-side evidence (firmware strings, live
telemetry) for hardware identification. See `docs/03_hardware.md` for what
is known and how, and `docs/18_future_work.md` Priority 2 for the
recommendation to add physical documentation in a future session.

Diagrams referenced from `docs/` are currently embedded inline as Mermaid
diagrams within the relevant Markdown files (e.g.
`docs/07_led_architecture.md`) rather than stored as separate image files
here — this keeps them version-controllable as text. If a diagram tool
that doesn't support inline Mermaid is used to read this repository,
rendered exports could be added here in the future.
